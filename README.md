## 1. Environment Setup

### Clone Repo

`$ git clone https://github.com/davies3matt/bounce.git`

### Enter Project Directory

`$ cd bounce`

### Run Package Manager

`$ yarn`

### Enter Backend Directory 

`$ cd backend`

### Run Package Manager & Build Strapi UI

`$ yarn && strapi build`

### Start Strapi

`$ strapi start`

### Enter Project Direcottry & Start bounce

`$ cd ..`
`$ yarn start`
